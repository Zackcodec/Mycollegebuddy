# 🎓 The Ultimate Campus Hub & Resource Workspace

A premium, glassmorphism-styled web workspace designed to eliminate academic friction and supercharge student life. Built using clean, responsive HTML5, modern CSS variables, and vanilla JavaScript.

## 🚀 Key Features

*   **⚡ The Resource Vault:** Instantly filter and download Previous Year Papers (PYQs), textbooks, and handwritten core subject notes.
*   **📊 Academic Tool Suite:**
    *   *Semester Target Calculator:* Know exactly what you need to score on your written theory papers based on your CCE internals and lab practicals.
    *   *CGPA/SGPA Converter:* Convert credit indexes to standardized linear percentages instantly using technical board formulas.
    *   *75% Attendance Runway & Priority Matrices:* Stay on top of your daily student tasks without burning out.
*   **🎉 Campus Life Feed:** A mobile-optimized, horizontal-scrolling event engine tracking upcoming cultural fests, department freshers, and hackathons.
*   **🤝 Transparent Support Hub:** An optimized, impact-tiered contribution panel with integrated regulatory disclosures for voluntary hosting maintenance.

## 🛠️ Tech Stack Overview

*   **Frontend Structure:** Semantic HTML5
*   **UI Elements:** Modern CSS3 (Glassmorphism Effects, Flexbox/Grid Matrixing, Custom Touch Scrolling)
*   **Logic Processing:** Vanilla JavaScript Core Engines (Zero external bulky dependencies)

## 💻 Local Installation Setup

To run this platform locally on your machine:
1. Clone the repository: `git clone https://github.com/Zackcodec/mycampusbuddy.git`
2. Open the root `index.html` file directly in any modern web browser.
